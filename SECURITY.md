# Security

## Поддерживаемые версии

Актуально поддерживается ветка **`main`** (текущий релиз шаблона).

## Сообщить об уязвимости

Не открывайте публичный issue с деталями уязвимости.

1. В репозитории на GitHub: **Security** → **Report a vulnerability** (если доступно), либо напишите владельцам организации приватно.
2. Укажите шаги воспроизведения, затронутые версии и, по возможности, минимальный PoC без публикации эксплойта.

Ответ обычно в течение нескольких рабочих дней. Критические находки обрабатываются в приоритетном порядке.
